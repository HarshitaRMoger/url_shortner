# URL Shortener

A full-stack URL Shortener application that converts long URLs into short, shareable links. The application generates unique short URLs and redirects users to the original URL efficiently.

## Features

- Convert long URLs into short URLs
- Unique URL generation
- Redirect short URLs to original URLs
- Store URL mappings in the database
- Responsive user interface
- Error handling for invalid URLs

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Installation

1. Clone the repository

```bash
git clone https://github.com/HarshitaRMoger/url_shortner.git
```

2. Navigate to the project directory

```bash
cd url_shortner
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
npm start
```

5. Open in browser

```
http://localhost:3000
```
